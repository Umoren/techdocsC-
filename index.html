Introduction

C++ is a general-purpose, case-sensitive, compiled, statically typed high-level programming language that supports object-oriented programming. It was developed by Bjarne Stroustrup in 1979.

This documentation provides an informal introduction to C++, covering basic concepts and syntax. We'll use Visual Studio as our compiler in the examples, which explains the #include <stdafx.h> at the start of every program (exclusive to Visual Studio users).

All C++ source file programs are saved with a .cpp (C++ preprocessor) file extension.

Your First C++ Program

The best way to learn a programming language is by writing programs. Typically, the first program beginners write is a "Hello World" program, which simply prints "Hello World" to your computer screen. Although simple, it contains all the fundamental components C++ programs have:

//my First C++ Program
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World" << endl;
    return 0;
}

Let's break down this program:

  • //my First C++ Program: This is a comment. Comments are ignored by the compiler and are used to explain the code.
  • #include <iostream>: This line tells the preprocessor to include the iostream header, which allows for input/output operations.
  • using namespace std;: This line allows us to use elements of the standard library without prefixing them with std::.
  • int main() { ... }: This is the main function, where program execution begins.
  • cout << "Hello World" << endl;: This line outputs "Hello World" to the console.
  • return 0;: This indicates that the program ended successfully.

Variables in C++

Variables are fundamental to programming. They allow us to store and manipulate data in our programs.

Let's consider an example:

a = 5;
b = 2;
a = a + 1;
result = a - b;

In this example, we're storing values in memory, modifying them, and performing calculations. Your computer can store millions of such values and perform complex operations with them.

We can define a variable as a portion of memory used to store a value.

Each variable needs a unique name (identifier) that distinguishes it from others. In the example above, we used a, b, and result as variable names.

Variable Declaration in C++

C++ is a strongly-typed language, requiring every variable to be declared with its type before its first use. This informs the compiler about the memory size to reserve and how to interpret the variable's value.

The syntax to declare a new variable in C++ is straightforward: write the type followed by the variable name. For example:

int a;
float mynumber;

These are two valid variable declarations. The first declares an integer variable named 'a', and the second declares a float variable named 'mynumber'.

You can declare multiple variables of the same type in a single statement by separating their identifiers with commas:

int a, b, c;

Here's a more comprehensive example:

//operating with variables
#include <iostream>
using namespace std;

int main() {
    // declaring variables:
    int a, b;
    int result;

    // Process
    a = 5;
    b = 2;
    a = a + 1;
    result = a - b;

    //print out the result:
    cout << result;

    // terminate the program
    return 0;
}

Variable Scope

The scope of a variable defines where in the program the variable can be accessed. In C++, a variable can have either global scope or local scope.

Global Variables

A Global Variable is declared in the main body of the source code, outside any function or block. It can be accessed from any part of the program.

Local Variables

A Local Variable is declared within the body of a function or block. It can only be accessed within that specific function or block.

Constants

Constants are fixed values that do not change during program execution. They are also called literals. Constants can be categorized into:

  • Numeric constants (e.g., 83, 0213, 0x4b)
  • Floating-Point constants (e.g., 3.14159, 314159E-5)
  • String Constants (e.g., "Hello Aces", "Alex for SUG Senate")
  • Boolean Values (true, false)

There are two ways to define constants in C++:

  1. Using #define preprocessor
  2. Using const keyword

Here's an example using #define:

#include <iostream>
using namespace std;

#define JAJA 20
#define NELLIE 20
#define NEXTLINE

int main() {
    int friends;
    friends = JAJA * NELLIE;
    cout << "Jaja and Nellie are friends and they are equal to" << friends;
    cout << NEXTLINE;
    return 0;
}

Datatypes

Data types, as the name implies, are the types of data we store in variables. The data type depends on the value of the variable. In C++, when you create a variable, you must specify its data type.

Some common data types in C++ include:

  • int: for integer values
  • float: for single-precision floating point numbers
  • double: for double-precision floating point numbers
  • char: for single characters
  • bool: for boolean values (true or false)
  • string: for sequences of characters

Control Structures

Control structures allow you to control the flow of your program's execution based on certain conditions. They are essential for creating dynamic and responsive programs.

The main types of control structures in C++ are:

  • If-else statements
  • Switch statements
  • Loops (for, while, do-while)

These structures can use simple statements (separated by semicolons) or compound statements (enclosed in blocks using curly braces {}).

Arrays

An array is a data structure that can store a fixed-size collection of elements of the same data type. It's a sequence of variables that share the same name and are referenced using an index.

For example, if you wanted to store the scores of the first 10 students on an attendance list, you could use an array named 'aces_score':

int aces_score[10];  // Declares an array of 10 integers

// Assigning values
aces_score[0] = 85;
aces_score[1] = 92;
// ... and so on

Arrays provide an efficient way to handle multiple related values under a single variable name.
